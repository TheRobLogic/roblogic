On-Site Setup

  1. Pack: Mac Mini, GL-B1300 router, USB drive (labeled CampPeach), EcoFlow + solar panel, ethernet cable, power cables
  2. Plug in Mac Mini and GL-B1300 router. Connect them with an ethernet cable (Mini to any LAN port on the GL-B1300).
  3. Power on both devices. Wait about 2 minutes for all services to start automatically.
  4. Connect your phone to CampPeach WiFi (password: robisthebest)
  5. Open http://camp.peach in your browser. Everything should just work.
  6. Print QR stickers beforehand so guests can scan to join.

How It Works

The Hardware

A 2014 Mac Mini running Ubuntu Server acts as the media hub. A GL.iNet GL-B1300 travel router creates the "CampPeach" WiFi network. An EcoFlow RIVER 2 Pro battery with a 220W solar panel keeps everything powered off-grid.

No Internet Needed

Everything runs locally. The router's DNS points all *.camp.peach addresses to the Mac Mini. Phones think they have internet because the server spoofs connectivity check responses for iOS, Android, and Windows.

Media Library

A 2.7TB USB drive holds movies, TV shows, music, audiobooks, ebooks, and Wikipedia. All media is served through dedicated apps: Jellyfin for video, Navidrome for music, Audiobookshelf for audiobooks, Kavita for ebooks, and Kiwix for offline Wikipedia.

AI-Powered Radio

Type a vibe on the landing page (like "chill acoustic" or "90s hip hop") and a custom radio station is created instantly from the music library. The station plays continuously through an Icecast stream. A small local AI model (Qwen 2.5) enhances the playlist in the background.

AI Chat

Open WebUI connects to a local Ollama instance running Qwen 2.5 1.5B. It's a small model (good for camp trivia, cooking tips, general Q&A) but runs entirely on the Mac Mini's CPU. First query after boot takes about 85 seconds to warm up.

Power Budget

The whole system draws about 50W idle. The EcoFlow RIVER 2 Pro (768Wh) provides roughly 15 hours of runtime, and indefinite with the solar panel in good sun.

Troubleshooting

"No Internet" warning on phone?

Dismiss it. The server spoofs connectivity probes so this shouldn't happen often, but some devices are persistent. Everything works without real internet.

Page won't load?

Make sure you're on the CampPeach WiFi. Type the full URL with http:// prefix (not https). Example: http://camp.peach

AI Chat is super slow?

The first query after boot takes ~85 seconds (loading the model into RAM). After that it's 5-7 tokens/second. This is normal for CPU-only inference.

USB drive disappeared?

The exFAT drive can disconnect if bumped. SSH in as camp@192.168.8.10 (password: 12345) and run: sudo mount -a then restart the affected containers.
